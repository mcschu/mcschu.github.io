License Server Change Instructions

Ansys#

License Information

Interconnect Port: 2325 | FlexNet Port: 1055 | Hostname: anderson11

IMPORTANT

INSTRUCTIONS MAY VARY BASED ON VERSION INSTALLED

  • Start -> All Programs -> ANSYS XX.X (Version) -> ANSYS Client Licensing (May be under the root ANSYS folder)
    • Right click [Client] ANSLIC_ADMIN Utility
    • Select Specify the License Server machine (Depending on version it may just show the server on the utility front page)
    • Highlight the Server that is in the list and click Edit Selected Server Machine
      • Verify the following settings:
      • ANSYS Licensing Interconnect port number (default is 2325): 2325
      • ANSYS FLEXlm port number (default is 1055): 1055
      • Select number of license server machines: 1-server
      • Hostname 1: anderson11
    • Click OK
    • Close and verify Ansys opens properly

Autocad#

Important

Start with the Environmental Variable change then move on to the next and if all fails then full uninstall and reinstall

Change Environmental Variable#

Set the license server information in ADSKFLEX_LICENSE_FILE environment variable: In the Windows Startup menu, right-click My Computer > Properties > Advanced System Settings. In the System Properties dialog box, Advanced tab, click Environment Variables. In the Environment Variables dialog box, under System Variables, click New. In the New System Variable dialog box, in the Variable Name field, enter ADSKFLEX_LICENSE_FILE. In the Variable Value field, enter < Server Name >. If using multiple servers, enter multiple server values. Separate each entry with a semi-colon (;), for example:

@SERVER1;@SERVER2;@SERVER3 Click OK to exit the dialog boxes. Note: Setting this variable also sets this path in the Registry. If this variable is deleted with the path, it will also need to be removed from the Registry.

If computers that have DNS resolution issues, enter an IP address instead of a PC name in step 5 and you may also add the TCP port number (usually 27000 or 2080) the license service is running. For example 2080@SERVER1;2080@SERVER2.

Clean the Registry Clear the ADSKFLEX_LICENSE_FILE registry key from old value. Click Start > Run. Type regedit and press Enter. Make a backup of the Registry. Click File > Export. Enter a path and filename for the backup and click Save. Locate the ADSKFLEX_LICENSE_FILE variables and remove the server name values from the ADSKFLEX_LICENSE_FILE fields. They may be in these locations: HKEY_CURRENT_USER\Software\FLEXlm License Manager HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\FLEXlm License Manager

The registry locations for Windows XP that need to be changed are as follows: HKEY_LOCAL_MACHINE\Software\Flexlm License Manager\ADSKFLEX_LICENSE_FILE HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Control\Session Manager\Environment\ADSKFLEX_LICENSE_FILE HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Environment\ADSKFLEX_LICENSE_FILE The registry locations for Windows Vista/Windows 7 that needs to be changed are as follows:

HKEY_CURRENT_USER\Software\Flexlm License Manager\ADSKFLEX_LICENSE_FILE HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Control\Session Manager\Environment\ADSKFLEX_LICENSE_FILE HKEY_LOCAL_MACHINE\SYSTEM\ControlSet002\Control\Session Manager\Environment\ADSKFLEX_LICENSE_FILE HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Environment\ADSKFLEX_LICENSE_FILE HKEY_USERS\S-1-5-21-?\Software\FLEXlm License Manager\ADSKFLEX_LICENSE_FILE

Note: This environment variables may need to be searched for.

To adjust the license server in LICPATH.LIC file The license server information can also be adjusted in the LICPATH.LIC file for the associated program. For multiple programs installed on a computer, you may need to do this multiple times. When you use this method, the server name is stored in a text file. Locate the LICPATH.LIC file in your program installation folder. 2016 and older: C:\Program Files\Autodesk\ program_name 2017, 2018 and 2019: C:\ProgramData\Autodesk\CLM\LGS\ Product_Key_201x.0.0.F 2020: C:\ProgramData\Autodesk\AdskLicensingService\ Product_Key_2020.0.0.F Make a backup copy of the file under a slightly different name. Open the original LICPATH.LIC file in a simple text editor (e.g.Notepad). Edit the LICPATH.LIC file and remove everything except the top 2 lines using the following syntax: SERVER YOUR_SERVER_NAME 000000000000 USE_SERVER Save the LICPATH.LIC file and verify that it doesn't pick up a hidden .txt extension (How to turn on hidden file extensions in Windows).

Additional notes for versions 2020 After changing the LICPATH as indicated above, do the following: Open an elevated Command Prompt. Run the below command: cd C:\Program Files (x86)\Common Files\Autodesk Shared\AdskLicensing\Current\helper\ AdskLicensingInstHelper change -pk prodkey -pv 2020.0.0.F -lm "" -ls "" -lt "" Replace the prodkey with the product key correspondent to your software. See here for a full list.

Change License Server by Version#

You have a problem that forces you to reset the license type. To change the license type, first change the software licensing option.

Causes: You changed the license type but didn't change the license setting in the software. For example, you may have changed from a perpetual license to a subscription license. Or you may have changed from a single-user (stand-alone) to a multi-user (network) license. Your subscription has expired. You may need to get a new subscription rather than autorenewing the old one. You used the software in trial mode and then got a subscription using a different email address. You signed in to the Autodesk desktop app with an email address different from the one you used to get a subscription. Matlab#

License Information

Hostname: anderson11 | Port: 27000**

  • Navigate to C:\Program Files\MATLAB\R20XX\licenses\
  • Right click and open network.lic with a text editor (notepad)
  • Change the first line to the following:
    • SERVER ANDERSON11.vmi.edu 00155D0414A5 27000
  • Save and exit and it should now point to the new license server
  • Open to verify

Mathcad#

License Information

Hostname: anderson11 | Port: 7788**

IMPORTANT

This is based on per user so must be done by the primary user of the asset

  • When you open Mathcad it will prompt you to open the Mathcad License Setup wizard, click YES.
  • Select I want to configure Mathcad to use an existing license file
  • Select Use a license server
    • Server name: anderson11
    • Server Port : 7788 (Default)
  • Open to verify
